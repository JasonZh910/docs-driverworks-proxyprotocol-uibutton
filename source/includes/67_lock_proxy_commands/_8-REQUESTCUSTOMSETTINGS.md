## REQUEST\_CUSTOM\_SETTINGS

This command results in the Protocol Notification [`CUSTOM_SETTINGS`][1] being sent from the Protocol to the Proxy.


### Name

`REQUEST_CUSTOM_SETTINGS ()`


### Parameters

`None`


### Returns

`None`
