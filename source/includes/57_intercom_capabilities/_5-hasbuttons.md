## has\_buttons

This capability specifies whether this device type has custom buttons. Its value is a Boolean string. For example: “True” or “False”


### Signature

`<has_buttons></has_buttons>`


| Parameter | Description |
| --- | --- |
| bool | True/False |


### Example

```xml
<capabilities>
   <has_buttons>True</has_buttons>
</capabilities>
```
