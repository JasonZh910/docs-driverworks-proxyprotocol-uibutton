## KEY\_PRESS

Specifies that a key (button) was pressed on the UI interface. But different from`Â BUTTON_PRESS`, the actual key name is passed rather than its ID.


| Parameter   | Description | Description                                                          |
| ----------- | ----------- | -------------------------------------------------------------------- |
| Key Name    | STR         | The single key that we should emulate a press for.                   |
| InterfaceID | STR         | A unique string to identify which interface is sending this command. |
