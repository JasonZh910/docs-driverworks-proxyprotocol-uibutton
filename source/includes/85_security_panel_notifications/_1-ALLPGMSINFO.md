## ALL\_PGMS\_INFO

Sends and XML string of all PGMs (relays) and respective PGM information.


| Parameter | Description |
| --- | --- |
| xml | XML data. See example. |


### Example

```xml
<pgms>
		<pgm>
			<id>1</id>
			<is_open>true</is_open>
		</pgm>
		<pgm>
			<id>2</id>
			<is_open>false</is_open>
		</pgm>
		<pgm>
			<id>3</id>
			<is_open>false</is_open>
		</pgm>
</pgms>
```
