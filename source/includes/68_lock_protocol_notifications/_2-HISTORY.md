## HISTORY

Sent in response to the command `REQUEST_HISTORY`.


### Name

`HISTORY ()`


### Parameter

`None`


### Returns

`None`


### Example

```lua
<lock_history>
	<history_item>
		<history_id>history-ID INT</history_id>
		<date>date STRING</date>
		<time>time STRING</time>
		<message>message STRING</message>
		<source>source STRING</source>
	</history_item>
</lock_history>
```
