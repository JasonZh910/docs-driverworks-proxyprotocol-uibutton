## JOIN\_GROUP

Received when the driver is added to a Load Group.

### Name

`JOIN_GROUP ()`

| Parameter | Type | Description                        |
| --------- | ---- | ---------------------------------- |
| GROUP ID  | STR  | Device ID of the Load Group driver |
| KEEP SYNC | STR  | 1 for True and 0 for False         |

### Returns

`None`
