## GROUP\_RAMP\_TO\_LEVEL

Received when group is ramping to level.

### Name

`GROUP_RAMP_TO_LEVEL ()`

| Parameter | Type | Description                 |
| --------- | ---- | --------------------------- |
| GROUP ID  | STR  | Driver ID of the Load Group |
| LEVEL     | STR  | Level to ramp               |
| TIME      | STR  | Time rate for ramping       |

### Returns

`None`
