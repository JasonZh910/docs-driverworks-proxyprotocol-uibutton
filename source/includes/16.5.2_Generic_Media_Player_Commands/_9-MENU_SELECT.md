## MENU\_SELECT

Command to select an item from the list of menu items returned with GET\_MENU.


### Name

`MENU_SELECT ()`


| Parameter | Description                         |
| --------- | ----------------------------------- |
| ITEM      | The id of the item selected         |
| TYPE      | The item type returned in the list. |
| TITLE     | The name of the item.               |


### SendToProxy

`None`
