## GET\_STATE

Command handled by the proxy driver.  Immediately returns a block of XML information to the caller. Reports the current state of the device.


### Name

`GET_STATE ()`


### Parameters

`None`


### Returns

`None`

### Example

```xml
<fan_state>
  <current_speed>3</current_speed>
  <is_reversed>false</is_reversed>
</fan_state>
```
