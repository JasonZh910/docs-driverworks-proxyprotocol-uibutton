## hold\_rate\_min

Ramping API that determines Minimum length of time (in milliseconds) that can be set for Hold Rate of a keypad button press. Defaults to 1000.


### Signature

`<hold_rate_min></<hold_rate_min>`


### Type

Integer


### Dynamic Capability

No


### Example

```xml
<capabilities>
    <hold_rate_min>1000</hold_rate_min>
</capabilities>
```
