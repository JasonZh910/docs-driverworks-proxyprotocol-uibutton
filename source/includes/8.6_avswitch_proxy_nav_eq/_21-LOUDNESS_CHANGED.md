## LOUDNESS\_CHANGED

Navigator EQ notification that the loudness has changed.


### Name

`LOUDNESS_CHANGED`


| Parameter       | Type | Description                                                      |
| --------------- | ---- | ---------------------------------------------------------------- |
| OutputBindingID | INT  | OUTPUT: OutputBindingID                                          |
|                 | BOOL | ENABLED: LOUDNESS: True for loudness on, False for loudness off. |


### Returns

`None`
